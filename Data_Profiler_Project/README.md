# 📊 Data Profiler Project (EDA + Data Analysis)

## 🚀 Overview

This project focuses on **Exploratory Data Analysis (EDA)** and **data profiling** using Python. It helps in understanding the dataset by analyzing distributions, detecting missing values, identifying outliers, and generating meaningful insights.

---

## 📂 Project Structure

* **CSV Dataset** → Structured tabular data for analysis  
* **JSON Dataset** → Semi-structured data handling  
* **Jupyter Notebook** → Complete EDA and profiling implementation  

---

## 🛠️ Technologies Used

* Python 🐍  
* Pandas  
* NumPy  
* Matplotlib  
* Seaborn  
* Scipy (for statistical analysis)  

---

## 📊 Features

* Data Cleaning & Preprocessing  
* Handling Missing Values  
* Statistical Analysis (Mean, Median, Std, etc.)  
* Data Visualization (Histogram, Boxplot, Q-Q Plot)  
* Correlation Analysis  
* Skewness & Kurtosis Check  
* Normal Distribution Verification  

---

## 📈 Key Insights

* Distribution of numerical features analyzed using histograms  
* Outliers detected using boxplots  
* Relationships between variables identified using correlation  
* Normality of data checked using Q-Q plots  
* Dataset quality evaluated through statistical measures  

---

## 📄 Output

* Graphical visualizations (Histograms, Boxplots, etc.)  
* Statistical summaries of dataset  
* Cleaned dataset ready for further analysis or machine learning  

---

## 📑 Output Reports

* `data_profile_report.html` → Comprehensive automated profiling report  
* Visual analysis outputs generated within the notebook  
* Summary statistics and insights derived from EDA  

---

## 💡 Conclusion

This project demonstrates how to perform **complete EDA and data profiling** using Python. It provides a strong foundation for data preprocessing and helps in preparing datasets for machine learning models.
